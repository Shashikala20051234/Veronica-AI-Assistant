# Veronica AI Voice Assistant

## Overview
Veronica is an AI-based desktop voice assistant developed using Python. It can perform basic system tasks and respond to voice commands.

## Features
- Voice recognition
- Executes system commands
- Automates simple tasks
- User-friendly interaction

## Technologies Used
- Python
- Speech Recognition
- Text-to-Speech (TTS)

## How to Run
1. Install required libraries
2. Run the main Python file
3. Give voice commands

- Add more intelligent responses
- Integrate with APIs
- Improve accuracy
